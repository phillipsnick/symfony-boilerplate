# Symfony API Boilerplate

Boilerplate setup to run a Symfony API application. 

## Getting Started

### Prerequisites 

- [Docker](https://www.docker.com/get-started/)
- [Symfony CLI](https://symfony.com/doc/current/setup/symfony_cli.html)

### Running locally

Basic setup

```
symfony serve
```
